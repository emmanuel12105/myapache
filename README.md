myapache
=========

Simple role to install, configure and enable apache on CentOS/Fedora/RHEL for Ansible examples.

Requirements
------------

Instance based on CentOS/Fedora/RHEL.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: servers
      roles:
         - myapache
